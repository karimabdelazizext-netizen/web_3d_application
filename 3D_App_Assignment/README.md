# KOKS Fragrance 3D App

This is a 3D Web Application for a premium fragrance collection built with Three.js and Bootstrap 5.

- **GitHub Repository:** [https://github.com/karimabdelazizext-netizen/web_3d_application/tree/main/3D_App_Assignment](https://github.com/karimabdelazizext-netizen/web_3d_application/tree/main/3D_App_Assignment)

## Description

This project features:
- Interactive 3D perfume bottle models created in Blender.
- 3D scene rendering using Three.js with OrbitControls for 360-degree viewing.
- Dynamically generated textures using the HTML5 Canvas API.
- A responsive user interface built with Bootstrap 5.
- Features like Wireframe toggling, material updates, and camera presets.
